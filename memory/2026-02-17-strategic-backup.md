## üß† Brainstorm Mode (V3)

### Concept
Multi-advisor discussion where:
1. **Moderator** frames the question
2. **3-4 advisors** respond with their expertise
3. **Synthesis** provides clear recommendation
4. **Action plan** with next steps

### Example Flow
```
User: "Should I price at $19 or $29?"

Moderator: Frames discussion from strategic, financial, marketing angles

Strategy: "$19 positions for early-stage founders ‚Äî our core market"

Finance: "At $19 vs $29, need 52% more customers. Volume matters now."

Marketing: "Grandfather early users, create urgency messaging"

Synthesis: "Launch at $19, plan increase to $29 at month 6"

Action Plan: Set price, create Founding Member tier, plan migration
```

### Features Added
- Live streaming (see each advisor respond in real-time)
- No advisor limit (was 4, now unlimited)
- 3-minute timeout for Ollama
- Save to new note / insert into current note / copy
